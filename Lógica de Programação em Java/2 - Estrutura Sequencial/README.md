## Estudos Sobre Estruturas Sequenciais
<br><br>
A estrutua sequencial nada mais é que uma estrutura lógica sequencial, ou seja em sequencia.

Assim como lemos um livro da esquerda para a direita e de de cima para baixo, o programa se lê da mesma forma.

Ex:

número A = x

número B = y

soma = x + y
<br><br>

desta forma estaria correta a lógica de estrurua sequencial.

Outro Ex:

soma = x + y

número A = x

número B = y
<br><br>

desta forma estaria quebrando a lógica sequencial do algoritmo, visto que não tem como fazer a soma se não informar a ele antes os números a serem somados.
<br><br>
Expressões Aritiméticas

Operadores:

( + ) Adição
  
( - ) Subtração
 
( * ) multiplicação
  
( / ) Divisão

( % ) Resto da divisão

Variáveis:

A variável serve para armazenar valores.

Ex:

nome = "Pedro"

Ou seja eu armazenei uma variável chamada "nome" com um valor que no caso esse valor é "Pedro".

 <tipo> <nome> = <valor incial>

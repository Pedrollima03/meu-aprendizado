## Introdução ao Curso lógica de Programação em Java
<br><br>
A lógica de programação é a base de conhecimento antes de começarmos a programar de fato, esse entendimento de lógica fará com que entendemos como funciona desde a escrita até o entendimento do seu programa rodar, passando pela sintaxe, algoritmo, léxica, software, hardware, IDE, e tudo mais. É a compreensão básica para começar o entendimento sobre programação. 
<br><br>
Algoritmo -> é uma sequência de instruções finitas para se resolver um problema.

Ex:

Problema: ir ao trabalho

Algoritmo: 
- acordar
- tomar banho
- se arrumar
- tomar café
- escovar os dentes
- pegar o transporte
- chegar ao trabalho

Perceba que até chegar ao trabalho foi seguido uma série de instruções até que em fim chegamos a solução do problema.
Para a lógica de programação aplicada ao programa e ao nosso dia é a mesma coisa, para se resolver um problema devemos programar, através de uma sequencia "lógica", para que a máquina entenda o nosso problema e a execute.
<br><br>
Com isso utilizamos máquinas, o computador, para se executar um programa. Mas para isso devemos usar uma linguagem de porgramação ja que o computador não compreende nossa língua como aprendemos em nossas vidas, assim como nós o computador também tem regras léxicas e sintáticas para se escrever um programa.

Através de uma IDE (Integrated Development Environment) que nada mais é que um sistema inteligente que traduz essas regras ao computador para poder rodar o programa, pois as máquinas tem como línguagem nativa os númeors binários 0 e 1 que chamamos de línguagem de baixo nível, já a IDE que utilizamos é uma línguagem de alto nível.

Neste meio temos diversas linguagens de programação, que talvez já conheçam, como python, Java, C, C++, PHP, JavaScript e muitas outras.

Com a IDE podemos criar através de algoritmos um porgrama que pela própria IDE nos permite executar em nosso próprio computador, através de uma construção de código fonte, depurações e testes.

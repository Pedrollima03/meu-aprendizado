## Introdução ao Curso lógica de Programação em Java
<br><br>
A lógica de programação é a base de conhecimento antes de começarmos a programar de fato, esse entendimento de lógica fará com que entendemos como funciona desde a escrita até o entendimento do seu programa rodar, passando pela sintaxe, algoritmo, léxica, software, hardware, IDE, e tudo mais. É a compreensão básica para começar o entendimento sobre programação. 
<br><br>
Algoritmo -> é uma sequência de instruções finitas para se resolver um problema.
